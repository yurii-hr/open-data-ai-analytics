# Система моніторинґу інфраструктури проєкту

Цей модуль забезпечує збір та візуалізацію метрик для контейнеризованого застосунку, розгорнутого на віртуальній машині в Microsoft Azure.


## Сервіси, що збирають метрики
1. **Node Exporter:** Збирає апаратні метрики віртуальної машини (CPU, RAM, диск).
2. **cAdvisor:** Збирає метрики споживання ресурсів кожним окремим Docker-контейнером.
3. **Prometheus:** Опитує вищезгадані сервіси  та зберігає часові ряди даних.
4. **Grafana:** Відповідає за візуалізацію даних із Prometheus.

## Відкриті порти
* `9090` — інтерфейс Prometheus (статуси цілей, сирі запити).
* `3000` — вебінтерфейс Grafana (дашборди).
* `9100` — Node Exporter (внутрішній збір метрик).
* `8080` — cAdvisor (внутрішній збір метрик контейнерів).

## Як відкрити Grafana
1. Перейдіть у браузері за адресою: `http://<IP-АДРЕСА_ВАШОЇ_VM_В_AZURE>:3000`.

## Створені панелі
На головному дашборді реалізовано 5 панелей згідно з вимогами:
1. Завантаження CPU на VM (Node Exporter).
2. Використання оперативної пам'яти (Node Exporter).
3. Кількість активних контейнерів (cAdvisor).
4. Ресурси контейнера веб-інтерфейсу (RAM).
5. Метрика HTTP-запитів до сервісу (Prometheus).
