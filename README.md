# Аналіз мережі зупинок громадського транспорту Львова

## Мета проекту
Провести аналіз просторового розподілу зупинок громадського транспорту у Львівській громаді, виявити ключові транспортні вузли та оцінити якість наявних відкритих даних.

## Джерело даних
Дані отримані з порталу відкритих даних: 
[Перелік зупинок громадського транспорту м. Львова](https://data.gov.ua/en/dataset/lviv_transport-stops)

## Завдання
1. За допомогою алгоритму кластеризації на основі координат (`lat`, `lon`) виділити основні транспортні райони міста.
2. Визначити топ-10 вулиць за кількістю зупинок (`addressThoroughfare`) для аналізу найбільш завантажених транспортних артерій.
3. Порівняти щільність зупинок у самому Львові та прилеглих населених пунктах за допомогою колонки `addressPostName`.


## Висновки
Дані середньої якості, аналіз проведено успішно.
