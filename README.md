# Open Data AI Analytics

Мікросервісний конвеєр для збору, перевірки, аналізу та візуалізації відкритих транспортних даних. 

## Інфраструктура

Проєкт підтримує автоматизоване розгортання в хмарі Microsoft Azure за допомогою Terraform. Сценарій самостійно створює необхідні мережеві ресурси, віртуальну машину (Linux Ubuntu) та запускає Docker-контейнери через `cloud-init`.

### Як розгорнути проєкт в Azure:

**Крок 1. Підготовка Cloud Shell**
1. Відкрийте портал [Azure](https://portal.azure.com/).
2. Запустіть **Cloud Shell**.

**Крок 2. Клонування репозиторію**
У терміналі виконайте команду:
`git clone https://github.com/yurii-hr/open-data-ai-analytics.git`

**Крок 3. Ініціалізація та запуск Terraform**
Перейдіть до теки з конфігураціями інфраструктури та застосуйте їх:
`cd open-data-ai-analytics/infra/terraform`
`terraform init`
`terraform apply -auto-approve`

**Крок 4. Перевірка працездатності**
Після успішного виконання команди `terraform apply`, у терміналі з'явиться вивід `public_ip_address`.
1. Скопіюйте цю IP-адресу.
3. Відкрийте у браузері адресу: `http://<ваш_public_ip_address>:8000`. Має відкритися веб-інтерфейс застосунку.

**Крок 5. Видалення ресурсів**
Щоб зупинити сервер та уникнути витрат коштів, обов'язково видаліть інфраструктуру:
Перебуваючи у теці `infra/terraform/`, виконайте:
`terraform destroy -auto-approve`
