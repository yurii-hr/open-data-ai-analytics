# Контейнеризація модулів проєкту за допомогою Docker

Цей проєкт реалізує мікросервісну архітектуру для пайплайну аналізу транспортних зупинок Львова за допомогою Docker та Docker Compose.

## Структура проєкту
* `data/` - містить вхідний файл `stops.csv`.
* `data_load/` - мікросервіс зчитування CSV та завантаження у БД.
* `data_quality_analysis/` - мікросервіс перевірки якости даних.
* `data_research/` - мікросервіс обчислення статистик та кластеризації (K-Means, DBSCAN).
* `visualization/` - мікросервіс генерації графіків та HTML-звіту.
* `web/` - мікросервіс для роздачі фінального вебінтерфейсу.

## Команди запуску
Для локального запуску всього комплексу сервісів виконайте команду в кореневій папці:
`docker compose up --build`

Для зупинки контейнерів:
`docker compose down`

## Перелік портів
* **8000** - Вебінтерфейс доступний за адресою `http://localhost:8000`
